# 💰 Dashboard de Controle de Gastos

Um sistema web simples e funcional para controle de despesas pessoais, com visualização em gráfico e armazenamento local.

---

## 🚀 Acesse o projeto online
👉 https://kodaaaa1.github.io/expenses-dashboard/

---

## 🎯 Funcionalidades

✅ Adicionar gastos  
- Nome (ex: gasolina, mercado)  
- Valor (R$)  
- Categoria  

✅ Listagem dinâmica de despesas  

✅ Remoção de gastos  

✅ Cálculo automático do total gasto 💸  

✅ Gráfico de gastos por categoria 📊  

✅ Armazenamento no navegador (localStorage)  

---

## 🛠️ Tecnologias utilizadas

- 🧱 HTML5  
- 🎨 CSS3  
- ⚙️ JavaScript  
- 📊 Chart.js
